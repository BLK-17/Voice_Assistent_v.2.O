# 🎤 Voice Assistant (Python + GUI)

A desktop voice assistant built using **Python**, featuring a graphical interface and speech recognition capabilities.
It can open websites, folders, tell time/date, perform web searches, and respond to voice commands.

---

## ✨ Features

✅ Speech Recognition (Microphone input)
✅ Text-to-Speech Responses
✅ GUI Interface using Tkinter
✅ Open Websites (YouTube, Gmail, Google)
✅ Open System Folders (Desktop, Downloads, Documents)
✅ Web Search via Voice
✅ Tell Current Time & Date
✅ Close Applications
✅ Multi-threaded Voice Listening
✅ Offline TTS using pyttsx3

---

## 🖥️ Technologies Used

* Python 🐍
* Tkinter (GUI)
* SpeechRecognition
* PyAudio
* pyttsx3 (Offline TTS)
* Webbrowser module
* Subprocess & OS modules

```

---

```
